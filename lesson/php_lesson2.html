５分でわかるPHP文法シリーズ

次はif文です。英語のifは、仮定法とかで使いますよね。「もし～なら・・・・」みたいに使ってた記憶があります。多分、そうですよね。プログラミングのif文もそんな感じです。結構簡単なので、身構えずに、ぼちぼち行きましょう。

if文

if文は、（）の中の条件が成り立つ場合に、｛｝の中の処理を実行します。

例題として、夜ごはんは、口が完全にジンギスカンの気分の人がいるとします。
もし、変数の中がジンギスカンなら、「うまい」と言わせましょう

            
                $dinner = 'ジンギスカン';
                if ($dinner == 'ジンギスカン') {
                    echo "・・・うまい！";
                }

喜んでくれたみたいですね。

えーと、それでは、さらに複雑にしてみましょう。if文は「もしも～であれば・・・それとも～であれば・・・いずれでもなければ・・・・」というように多岐分岐が可能なので、複雑な心境も表すことができます。

たとえば、先ほどの人を例に出しましょう。口が完全にジンギスカンの気分の人は「もし夜ご飯がジンギスカンであれば」うまいと呟きましたよね

                    
                        $dinner = 'ジンギスカン';
                        if ($dinner == 'ジンギスカン') {
                            echo "・・・うまい！";
                        }

しかし、今の時間は23時半だとしましょう。疲れた彼には、ジンギスカンのお店を探す体力は残っていませんでした。駅を降りると、チェーンの牛丼屋が見えました。

「妥協することも大事」と、自分に言い聞かせるでしょう。

                    
                        $dinner = '牛丼';
                        if ($dinner == 'ジンギスカン') {
                            echo "・・・うまいな";
                        } elseif($dinner == '牛丼') {
                            echo "妥協することも大事";
                        }

つい先月、彼は先祖代々続いてきた家具屋さんを机と椅子以外売らないというポリシーを捨てることができず、つぶしてしまいました。牛丼屋に入ってもお金が勿体ないだろうと、彼は牛丼屋には入らずに、家路を急いだとしましょう。家に帰ると、既に妻と娘は寝ていました。明かりをつけると、机に冷え切ったカレーがポツンと置かれ、「お父さん、いつもありがとう」と書かれた手紙とプレゼントが置かれていました。今日は父の日だったのです。

この時の彼の感情はどうなるでしょうか

        
        $dinner = '家族の温かさと冷え切ったカレーライス';
        if ($dinner == 'Genghis Khan') {
            echo "・・・うまいな";
        } elseif ($dinner == '牛丼') {
            echo "妥協することも大事";
;        } else {
            echo "・・・・";
        }

「いずれでもなければあまりの嬉しさに言葉を失う」になりますね。

このように、複雑なものを表す時に必須の文法なので、覚えておきましょう。（ふざけてすみませんでした。まじめな説明を見たい方はこちらがおすすめです。英語ですが、プログラミングの勉強は基本、英語のドキュメントのほうが最新版であったり、正確である場合が僕の体感としては多いと思います。）
