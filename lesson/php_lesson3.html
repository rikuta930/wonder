５分でわかるPHP文法シリーズ

次は繰り返しを学びましょう。ここで覚えなければいけないのは、while文　for文　foreach文です。これに関しては、例文を乗せるので、反復で覚えるのがもっとも効果的でしょう。難しくはないです。ただ、ある条件を何度も繰り返すというだけなのです。

繰り返し

まずは、for文を学びましょう。ここでは、１から１００までを順番に出力しています。 

                    
                        for($i=1;$i<=100;$i++){
                            echo $i;
                        }

$i++　というのは、$i = $i+1という意味です。このように、略し方もいろいろあるので、ちょっとずつ覚えておきましょう。

次は、whileです。forと全く同じ動きをするので、最初はどちらかを覚えればよいと思います。 

                            
                                $i = 1;
                                while($i<=100) {
                                    echo $i;
                                    $i++:
                                }

次は、foreach文です。ここでは、配列を使うのですが、これは、変数の箱よりもさらに大きな箱を用意していると考えるとわかりやすいでしょう。 

                            
                                $scores = array('数学' => 60, '英語' => 40, '国語' => 20);
                                foreach($scores as $score) {
                                    echo $score;
                                }

さて・・、めちゃくちゃ駆け足に説明しましたが、これは、急に説明が面倒になったということではなく、繰り返しに関してはなんども使うことになるのですが、少し応用的な技術と組み合わせることが多いので、ある程度網羅的に理解しておいて、使って覚えることを推奨します。これで、PHPの文法編は終了です。

ここまで、頑張っていただき、ありがとうございます。慣れていない方は、非常に疲れたのではないでしょうか？プログラミングは、覚える必要はありませんが、理解する必要があります。このほかにもいくつか理解しておくべきものはありますが、やりながら、少しずつ慣れていきましょう！お疲れさまでした！
